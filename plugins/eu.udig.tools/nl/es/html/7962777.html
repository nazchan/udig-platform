Herramientas de Edición

Vista General de las Herramientas de Edición

Esta es la página raiz de la documentación para usuarios de las extensiones Editing Tools de uDig. Axios las provee bajo el proyecto Spatial Operations and Editing Tools promovido por la Diputación Foral de Gipuzkoa, Departamento de Movilidad y Ordenación del Territorio.

Estas extensiones integran algunas herramientas adicionales para la edición de features basados en geometrías.

Revisa los siguientes enlaces para la documentación de usuario específica para cada una de las extensiones disponibles.

  • Herramienta Ajuste de Líneas — Ajusta LineStrings hasta un punto usando una línea de ajuste
  • Herramienta Arco — Crea un linestring que se asemeja a un arco dados tres puntos de control (punto inicial, eje y punto final).
  • Herramienta Dividir — Divide features cruzados por una línea
  • Herramienta Fusión — Fusiona geometrías y atributos de varios Features
  • Herramienta Paralela — Crea un LineString paralelo.
