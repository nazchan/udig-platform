Download the package appropriate to your platform, unzip and run:

  • Windows
  • Linux
  • Mac

Windows and Linux packages includes a Java Runtime so you really can unzip and run without making any configuration changes to your Machine. On Mac OSX uDig is able to check for Java Advanced Imaging and install if needed.

About uDig 1.3.2

The 1.3.2 is a minor patch release showcasing active assistance when writing CQL expressions and filters. If all goes well this technology will be used to making editing styles a more rewarding and dynamic experience.

This release has been generated with maven using the tycho plugin. This technology will allow us to issues on a more frequent basis. Thanks to Cliff Broadbent for the research and effort required to set this up.

The 1.3 series features some great usability improvements with a new tool palette and easy to access tool options.

For developers this release of the uDig platform provides access to the latest open source libraries.

Here are some new features you can look forward to:

  • Active assistance when editing CQL Filters and Expressions
  • Area - define your area of interest based on CRS, current screen or a bookmark
  • Filter the Table view using the current area of interest - a great way to focus on what is on screen
  • Updated Info and Profile tool for raster data
  • Teradata support
  • Spatial Toolbox view allowing access to OMS3 processing; with tutorial!
  • New import and export formats: CSV and KML.
  • Checkout What is new page for details

Usability Improvements

Known Issues:

  • The mac release is unable to support the full range of ImageIO-EXT formats. Please contact udig-devel if you are able to build GDAL on Mac OSX.
  • The ImageIO-EXT / GDAL library requires native code for MrSID and ECW available on limited platforms (win32, linux32)
